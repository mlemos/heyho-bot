# HeyHo Bot: AI-Powered Investment Research

## The Problem
VC Associates spend 2-4 hours manually researching each investment opportunity:
- Searching multiple sources for company information
- Tracking down founder backgrounds
- Analyzing funding history and investors
- Evaluating competitive landscape
- Writing investment memos

## The Solution
HeyHo Bot automates the entire research-to-memo pipeline in under 2 minutes.

## How It Works

### 1. Smart Input Processing
- **Drop any files**: Pitch decks, financial docs, images, audio, video
- **Or just type**: Company name or URL
- **AI Triage**: Automatically classifies and extracts relevant data from files

### 2. Parallel Research Engine
8 specialized AI agents research simultaneously:
- Company Basics
- Founder Profiles
- Funding History
- Product Analysis
- Traction Metrics
- Market Potential
- Competitive Landscape
- Recent News

### 3. Intelligent Synthesis
- Combines uploaded file data with live web research
- Structures findings into comprehensive analysis
- Scores across 5 dimensions (Team, Market, Product, Traction, Competition)

### 4. Investment Memo Generation
Professional output includes:
- Executive Summary
- Company Scorecard (0-10)
- Fund Fit Analysis
- Strategic Partner Matching
- Risk Assessment
- Cited Sources

### 5. CRM Integration
- Auto-saves to Attio CRM
- Creates company and opportunity records
- Links to original sources

## Key Features

### Multi-Modal Intelligence
- Processes PDFs, images, audio, video
- Extracts data from pitch decks automatically
- Understands financial documents and cap tables

### Real-Time Progress
- Live pipeline visualization
- Token usage tracking
- Processing time display

### Customizable Scoring
- Configure fund thesis alignment
- Define strategic partner profiles
- Set stage and sector preferences

### PWA Ready
- Works offline (cached)
- Install on iOS/Android home screen
- Dark/Light mode support

## Results

| Metric | Before | After |
